DevOps Agent — 前端设计文档

采用 明日方舟（Arknights）莱茵生命（Rhine Lab）PV 视觉风格——科技感、工业风、冷色调的运维监控界面。

视觉要素

要素应用
六边形网格背景 Canvas 动画，分子结构般的六边形网络
扫描线全屏 CSS 伪扫描线效果，模拟监视器画面
几何图形六边形 Logo、面板标记、架构图节点
冷色调蓝色 #3b82c4、青蓝 #2d9d8e、冰蓝 #7ec8e3 为主
橙色点缀警告/高风险使用琥珀色 #e8a838
等宽字体数据展示、终端输出统一使用等宽字体
发光效果边框发光(glow)、按钮波纹、节点脉冲动画
工业感2px 细线分隔、直角矩形为主

配色方案

变量色值用途
--rhine-blue #3b82c4主色调 — 莱茵蓝
--rhine-teal #2d9d8e辅助色 — 成功/正常
--rhine-ice #7ec8e3辅助色 — 信息
--rhine-dark #1e2d3d深色文字
--rhine-warn #e8a838警告 — 琥珀色
--rhine-danger #d04040危险/严重

动画系统

动画名元素效果周期
hex-rotateLogo 六边形旋转 + 发光脉冲20s
scanline-drift全屏扫描线垂直漂移8s
status-blink状态指示器呼吸灯2s

前端技术

技术说明

Canvas六边形背景动画（分子网格）
